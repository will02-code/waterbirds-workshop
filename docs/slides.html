Model Building

Model Structure

Hurdle-Gamma Model for zero-inflated count data:

  • Hurdle component — Binomial model for presence/absence
  • Gamma component — Continuous positive values when present
  • Bayesian estimation — Full uncertainty quantification via MCMC

Environmental Predictors

Flow - Wetland-level annual flow - Basin-level flow (3-month) - Lagged effects

Precipitation - Asset-level rainfall - Valley-level rainfall - Annual totals

Surface Water - Wetland inundation - Basin inundation - Valley inundation - Lagged conditions

Model Fitting

Code 
m1.1 <- brm(
  full_formula,
  family = hurdle_gamma(),
  data = full_data,
  backend = "cmdstanr",
  cores = 4,
  chains = 4,
  iter = 4000
)

Model Diagnostics

Check convergence and fit:

  • Trace plots — MCMC chain mixing
  • R-hat values — Should be < 1.01
  • Posterior predictive checks — Compare simulated to observed data
  • Bayesian R² — Variance explained
