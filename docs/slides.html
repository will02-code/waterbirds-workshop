Trend analysis

eawsAnalysis package

This package is a collections of functions under to interrogate the waterbird database. We’ll run through some here:

  • clean_data_run_models: Main function which cleans data, runs models, and cleans output.
  • run_mk_models: Side-car modeling function.
  • fit_mk_model: Fit Mann-Kendall model to a single group of data.
  • clean_clip_raw_data: Clip and clean raw waterbird data.
  • get_bird_data: Extract cleaned waterbird data.
  • clean_model_output: Clean output from run_mk_models().
  • post_fitting_line_plot: Plot model results with ggplot2

scaling <- combined_flow_metadata |>
    filter(wetlands == wetland) |>
    slice(1) |>
    select(
      wetland_mean_flow_3m,
      wetland_sd_flow_3m,
      wetland_mean_flow_annual,
      wetland_sd_flow_annual
    )

scenarios <- valley_climate_projections_with_wetlands |>
    filter(Wetland == wetland) |>
    arrange(Scenario) |>
    select(Scenario, pct_change) |>
    left_join(
      basin_flow_mean_sd |>
        select(Scenario, basin_flow_zscore_3m, basin_flow_zscore_annual),
      by = "Scenario"
    ) |>
    mutate(
      wetland_flow_3m_scaled = (scaling$wetland_mean_flow_3m * pct_change) /
        scaling$wetland_sd_flow_3m,
      wetland_flow_annual_scaled = (scaling$wetland_mean_flow_annual *
        pct_change) /
        scaling$wetland_sd_flow_annual,
      wetland_flow_annual_scaled_lagged = wetland_flow_annual_scaled,
      total_basin_flow_3m_scaled = basin_flow_zscore_3m,
      total_basin_flow_annual_scaled = basin_flow_zscore_annual,
      total_basin_flow_annual_scaled_lagged = basin_flow_zscore_annual,
      Wetland = wetland
    ) |>

Environmental water analysis

Overview

  • This is very similar to the climate analysis, but we use the model to predict abundance as if environmental flows had not existed previously
  • Again, I’m going to skip the cleaning here - we can work through the website if we have time

Cleaned data

  • Ultimately we end up with an annual dataset that has two possible columns for total flow at the upstream gauge - one column has actual, and one has actual minus environmental flows.
